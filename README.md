# DungeonOfDoom
Simple game created for a coursework!
Objective: Collect gold and escape!
Instructions: 
## Movement
'n' - Go north \
'e' - Go east \
's - Go south \
'w' - Go west 
 
# Commands
'hello' - Displays gold needed to win \
'gold' - Gold currently owned \
'pickup' - Attempt to pick up gold on current space \
'look' - Look around you \
'quit - Quits game
